Strolling Around

An Android application aiming to provide Accessible Navigation services to people with mobility disabilities. Built using Google Maps API and crowdsourcing to share pins noting accessible entrances, ramps, elevators, etc.


vlogMe

A Web App that allows users to upload Video Topics and the ability to share Video Responses to those topics. The UI allows for easy switching between the Main Topic and relevant Responses at specific points of the Main Video. Developed using the Polymer 1.0 framework, DASH video, and video.js.


Living In Technology (LIT)

An affordable SmartHome system, Living In Technology provides a framework for controlling devices at home from anywhere in the world through its Web Services. Hardware built in a cost-effectie manner through the use of Raspberry Pi devices with the LIT software included. Making use of Apache Web servers, SpringBoot RESTful services, Pi4j Java library, and other open-source software to build a comprehensive and responsive service. The System is currently in its prototype stages, and will be released at CSUN's School of Engineering and Computer Science 2017 Senior Design showcase
